React Hotkeys Docs
==================

Docusaurus site for documenting `@selokhq/react-hotkeys`, including live examples and autogenerated API/type references pulled straight from the library source.

Installation
-------------
```sh
npm install
```

Local Development
---------------
```sh
npm start
```

Type/API docs generation
------------------------
- A custom plugin in `src/plugins/generate-docs` runs on `start`/`build`, reading `packages/react-hotkeys/src/lib/types` and emitting MDX pages under `docs/Types` (see `docs/types/GENERATED.md`).
- Edits to generated type pages should be made in the library JSDoc/type definitions; the plugin will regenerate the MDX on the next run.

Content layout
--------------
- `docs/` holds authored pages (getting started, defining hotkeys, useHotkey, key types) and live-code snippets under `docs/assets/`.
- `src/components` and `src/utils` house docs-only helpers like the live editor and type declaration renderer.
- Static assets live in `static/` (favicons, images, keyboard layout PDFs).
